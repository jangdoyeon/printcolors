printcolors
=========
class for printing color text in terminal

class code
---------
```
class Pcolors:
    RED = '\033[95m'
    BLUE = '\033[94m'
    GREEN = '\033[92m'
    YELLO = '\033[93m'
    ORANGE = '\033[91m'
    GRAY = '\033[0m'
    BOLD = '\033[1m'
    UNDERLINE = '\033[4m'


print(f"""
{Pcolors.RED}RED
{Pcolors.BLUE}BLUE
{Pcolors.GREEN}GREEN
{Pcolors.YELLO}YELLO
{Pcolors.ORANGE}ORANGE
{Pcolors.GRAY}GRAY
{Pcolors.BOLD}BOLD
{Pcolors.UNDERLINE}UNDERLINE
""")
```
